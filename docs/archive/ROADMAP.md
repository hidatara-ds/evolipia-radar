# Evolipia Radar Roadmap

## Completed Phases
- **Phase 1: Core AI Gateway** - Centralized prompt proxy to Go.
- **Phase 2: Semantic Pipeline** - pgvector similarity clustering, AI-titling, strict ranking.
- **Phase 3: Intelligent Orchestration** - Multi-agent discover bots (RSS, Trending), zero-cost API ingestion.
- **Phase 4: DevOps & Safety** - Token budget protections, rate-limit fallback clustering, Vercel CI bounds.
- **Phase 4.5: Operations Dashboard** - Minimalist frontend tracking system metrics and manual overrides.

## Upcoming Phases

### Phase 5: Deep Intelligence Engine
- Migrate clustering logic out of `internal/ai/` into `internal/cluster/`.
- Introduce full semantic vector database activation.
- Activate long-term cluster merging and parent-child hierarchical grouping.

### Phase 6: Multi-Channel Delivery
- Web client rich UI (replacing MVP dashboard).
- Flutter mobile client bindings.
- WhatsApp / Telegram bot delivery based on virality triggers.
