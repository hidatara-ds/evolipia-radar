## Mulai dari mana?

- **Belum punya Go?** → [INSTALL_GO_WINDOWS.md](INSTALL_GO_WINDOWS.md)
- **Mau lihat dependency?** → [DEPENDENCIES.md](DEPENDENCIES.md)
- **Mau baca rencana fitur ke depan?** → [ENHANCEMENT_PLAN.md](ENHANCEMENT_PLAN.md)
