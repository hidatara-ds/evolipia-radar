# 📱 Cara Install Aplikasi di HP

Evolipia Radar bisa di-install di HP seperti aplikasi native, tanpa perlu download dari Play Store atau App Store!

---

## 🤖 Android (Chrome)

### Cara 1: Install Otomatis
1. Buka **Chrome** di HP
2. Kunjungi URL aplikasi (misal: `https://evolipia-radar.fly.dev`)
3. Banner "Add to Home Screen" akan muncul di bawah
4. Tap **"Install"** atau **"Add"**
5. Selesai! Icon muncul di home screen

### Cara 2: Manual
1. Buka **Chrome** di HP
2. Kunjungi URL aplikasi
3. Tap menu **⋮** (3 titik) di pojok kanan atas
4. Pilih **"Install app"** atau **"Add to Home Screen"**
5. Tap **"Install"**
6. Selesai!

### Cara Buka:
- Tap icon **Evolipia Radar** di home screen
- App buka fullscreen (tanpa browser bar)

---

## 🍎 iOS (Safari)

### Langkah Install:
1. Buka **Safari** di iPhone/iPad (harus Safari, bukan Chrome)
2. Kunjungi URL aplikasi
3. Tap tombol **Share** (⬆️) di bawah
4. Scroll ke bawah, pilih **"Add to Home Screen"**
5. Edit nama jika perlu
6. Tap **"Add"** di pojok kanan atas
7. Selesai!

### Cara Buka:
- Tap icon **Evolipia Radar** di home screen
- App buka fullscreen

## 🔧 Troubleshooting

### "Add to Home Screen" tidak muncul (Android)
- Pastikan buka di **Chrome** (bukan browser lain)
- Pastikan URL pakai **HTTPS** (ada gembok 🔒)
- Coba refresh halaman (swipe down)
- Coba clear cache Chrome

### "Add to Home Screen" tidak ada (iOS)
- Harus buka di **Safari** (bukan Chrome iOS)
- Pastikan URL pakai **HTTPS**
- Coba buka di tab baru (bukan private browsing)

### App tidak buka fullscreen
- Uninstall app dari home screen
- Install ulang dari browser
- Pastikan buka dari icon (bukan bookmark)

### App lemot atau error
- Cek koneksi internet
- Clear cache app:
  - Android: Settings > Apps > Evolipia Radar > Storage > Clear Cache
  - iOS: Settings > Safari > Clear History and Website Data
- Reinstall app
